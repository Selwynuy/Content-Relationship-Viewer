A dynamic carousel plugin for college departments with background image transitions and Elementor integration.

== Description ==

College Department Carousel is a powerful WordPress plugin that creates beautiful, responsive carousels for college departments. It integrates seamlessly with Elementor and Advanced Custom Fields (ACF) to provide a flexible and easy-to-use solution for displaying department information.

= Key Features =

* Responsive carousel with Swiper.js
* Dynamic background image transitions
* Elementor widget integration
* ACF field support for content
* Customizable slide settings
* Mobile-friendly design
* Touch-enabled navigation
* Automatic and manual navigation options

= Requirements =

* WordPress 5.0 or higher
* Elementor 3.0 or higher
* Advanced Custom Fields (ACF) 5.0 or higher

== Installation ==

1. Upload the `college-department` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create ACF fields for your carousel content
4. Add the carousel widget to your Elementor page

== Frequently Asked Questions ==

= How do I set up the carousel? =

1. Create ACF fields for your carousel content
2. Add the carousel widget to your Elementor page
3. Configure the settings in the Elementor panel
4. Add your content using the ACF fields

= Why do I need to set slides to show as slides + 1? =

The carousel uses a loop feature for smooth transitions. Setting the number of slides to show as your desired number plus one ensures proper spacing and centering of the active slide. For example, if you want to show 3 slides, set "Slides to Show" to 4.

= How do I add background images to slides? =

1. Add a background image field in ACF
2. Set the field name to "background_image"
3. The carousel will automatically use this image for the slide background
